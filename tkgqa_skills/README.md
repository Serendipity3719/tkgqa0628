# TKGQA Skill System (Phase 1)

This directory introduces the Phase 1 Knowledge Skill Abstraction layer for TKGQA.

## Concept
Skills are no longer tools (grep/find/awk), but semantic navigation units:

- Entity Skills
- Relation Skills
- Temporal Skills
- Event Cluster Skills

This is the first step toward CORPUS2SKILL-style hierarchical navigation for Temporal Knowledge Graph QA.
